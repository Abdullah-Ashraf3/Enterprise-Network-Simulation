\# Enterprise Network Simulation



This repository contains a Cisco Packet Tracer simulation for a multi-site enterprise network. It demonstrates WAN connectivity and routing between a Headquarters (HQ) and two remote branches through an Internet Service Provider (ISP). The repository also includes startup and running configuration backups for the routers and switches used in the network.



\## Project Overview



The project is designed to simulate a real-world enterprise network environment. It shows how multiple sites can communicate through routed WAN links, making it useful for learning network design, routing, switching, and device configuration.



The topology includes:



\- Headquarters site

\- Branch A site

\- Branch B site

\- ISP connection between sites



\## Features
